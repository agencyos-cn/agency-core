# 贡献者公约

## 我们的承诺

身为项目成员、贡献者和负责人，我们承诺使参与我们的社区成为每个人的无骚扰体验，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育背景、社会经济地位、国籍、个人形象、种族、宗教或性认同和性取向如何。

我们承诺以促进开放、友好、多元、包容和健康社区的方式行事和互动。

## 我们的标准

有助于在本社区创造积极环境的行为包括：

* 表现出对他人的同情和善意
* 尊重不同的意见、观点和经验
* 给予并优雅地接受建设性反馈
* 承担责任并向受我们错误影响的人道歉，从经验中学习
* 专注于不仅对我们个人，而且对整个社区最有利的事情

不可接受的行为包括：

* 使用 sexualized 语言或图像，以及任何形式的性关注或性挑逗
* 挑衅、侮辱或贬低性评论，以及个人或政治攻击
* 公开或私下骚扰
* 未经明确许可，发布他人的私人信息，如物理地址或电子邮件地址
* 其他在专业环境中被合理认为不恰当的行为

## 责任与义务

项目负责人有责任澄清可接受行为的标准，并应对任何不可接受的行为采取适当和公平的纠正措施。

项目负责人有权和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 和其他贡献，并会出于他们认为合适的原因暂时或永久禁止任何贡献者参与其他行为，他们认为这些行为是不适当的、威胁性的、冒犯性的或有害的。

## 范围

本行为准则适用于所有社区空间，也适用于个人在公共场所正式代表社区时。代表我们社区的例子包括使用官方电子邮件地址、通过官方社交媒体账户发布信息，或在线上或线下活动中担任指定代表。

## 执行

如有滥用、骚扰或其他不可接受的行为，可通过项目邮箱 **contact@agencyos.cn** 向项目负责人报告。所有投诉将得到及时和公平的审查和调查。

项目负责人有义务尊重任何事件报告者的隐私和安全。

## 执行指南

项目负责人将遵循以下社区影响指南，确定他们认为违反本行为准则的行为的后果：

### 1. 纠正

**社区影响**：使用不恰当的语言或其他被认为不专业或不受欢迎的行为。

**后果**：来自项目负责人的私人书面警告，明确说明违规行为的性质，并解释行为为何不恰当。可能会要求公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行动造成的违规行为。

**后果**：警告并停止行为的后果。否则可能导致临时或永久封禁，包括在指定时间内不得与相关人员（包括未经请求的互动）互动。这包括避免在社区空间以及外部渠道（如社交媒体）的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内，不得与社区进行任何形式的公共或私下互动。在此期间，不得与被临时封禁的人进行公开或私下互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：显示出一种违反社区标准的模式，包括持续的不当行为、骚扰个人或对某些人群体进行攻击。

**后果**：永久禁止在社区内进行任何形式的公共互动。

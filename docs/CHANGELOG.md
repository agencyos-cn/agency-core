# 变更日志

所有项目的显著变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [0.2.0] - 2026-03-18
### 新增
- 实现角色系统，支持创建不同专业领域的AI角色（教师、律师、金融顾问、个人助理等）
- 添加角色管理API，支持角色的创建、更新、删除和切换功能
- 实现Web API模块，提供RESTful接口支持前端管理
- 添加角色特定的AI模型配置功能
- 实现角色权限控制系统，限制可执行的技能范围
- 添加角色人格特质和沟通风格配置
- 创建Web服务器启动脚本
- 添加全面的测试用例，包括角色管理和Web API测试
- 新增 Dify 工作流集成，支持外部编排能力
- 接入 OpenClaw 智能体框架，扩展多模态交互能力
- 对接主流大模型平台（如 OpenAI、火山方舟等），统一模型调用接口

### 变更
- 扩展RuntimeContext以支持角色ID
- 意图引擎根据角色类型进行领域特定的意图解析
- 技能编排器基于角色生成专业化的执行计划
- 配置管理系统支持角色级别的配置隔离
- 重构配置体系，拆分多用途配置文件
- 按功能域组织配置，提升可维护性与扩展性
- 统一配置加载逻辑，支持环境变量与配置文件混合读取
- 重构字符管理功能，支持专业角色管理

### 修复
- 修复配置管理器中缺失的方法
- 修复导入错误和模块依赖问题

### 安全
- 实现角色权限控制，限制角色能力边界

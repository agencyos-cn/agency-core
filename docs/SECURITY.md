# 安全策略

## 支持的版本

由于项目目前处于积极开发阶段（v0.x），我们只对最新版本提供安全更新。

| 版本 | 支持状态 |
|:---|:---|
| 0.x | ✅ 积极开发中 |
| < 0.1 | ❌ 不再支持 |

## 报告漏洞

我们非常重视安全问题。如果你发现安全漏洞，**请不要在公开的 Issues 中报告**。

### 报告方式

1. **邮箱**：发送邮件至 `contact@agencyos.cn`
2. **PGP 加密**：（可选，如果你有 PGP 公钥，可以添加）

### 报告内容请包含

- 漏洞类型（如：权限提升、信息泄露等）
- 完整复现步骤
- 受影响的版本
- 可能的修复建议（可选）

### 处理流程

1. 我们会在 48 小时内确认收到报告
2. 评估漏洞严重程度和影响范围
3. 开发修复方案
4. 发布安全更新
5. 公开致谢（如果你同意）

我们承诺在修复前不会公开漏洞细节。

# AgencyOS 架构设计

## 整体架构

AgencyOS 采用端-边-云连续体架构，分为四大执行域：

### 1. 个人计算域
- 设备类型：电脑、手机、平板
- 核心能力：数字世界操作
- 典型场景：文档处理、邮件发送

### 2. 可穿戴感知域
- 设备类型：智能眼镜、耳机、手表、手环
- 核心能力：人体感知、自然交互
- 典型场景：AR导航、健康监测

### 3. 具身智能域
- 设备类型：四足/人形机器人、机械臂
- 核心能力：物理移动、物体操作
- 典型场景：取快递、叠被子

### 4. 泛在物联域
- 设备类型：智能家居、传感器、基础设施
- 核心能力：环境控制、状态监测
- 典型场景：智能家居、智慧城市

## 行为单元四层模型

- **任务 (Task)**：顶层用户目标
- **技能 (Skill)**：可复用行为单元
- **服务 (Service)**：运行时功能组件
- **原语 (Primitive)**：硬件原子操作
